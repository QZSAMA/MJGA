# 开发路线图

## 阶段一: 原型验证 (当前阶段)

- [x] 创建项目基础结构，Ant 构建脚本
- [x] 实现基础 HTTP 请求客户端
- [x] 简单 JSON 请求/响应解析
- [x] 基本文字输入界面 (Form + TextField + 折叠/展开)
- [ ] 在 W995 模拟器中测试联网
- [ ] 真机安装测试 basic ping 请求

**完成目标**: 能从 W995 成功发送文字到 OpenClaw 服务器并收到响应显示在手机上。

**已知问题**:
- `build.xml` 中 `javac target` 设为 1.8，应改为 1.6/1.7 以符合项目规范
- `HttpClient` 逐字节读取响应，UTF-8 多字节中文会乱码
- `JsonParser` 解析转义引号 `\"` 时会截断
- `res/` 目录缺失，Ant 构建的 `resources` target 会报错
- 模拟器缺少 `microemu-device-large.jar`，`run-emulator.sh` 无法运行
- JAD 文件缺少 `MIDlet-Jar-Size` 字段

## 阶段二: 功能完善

- [ ] 历史消息记录
- [ ] 可配置服务器地址 (不用硬编码)

## 阶段三: 服务端适配

- [x] OpenClaw 服务端 API 端点 (Go + Python 双版本代理已实现)

**已知问题**:
